# AI Pulse — 90초 데모 영상 대본 (RECORDING.md)

읽기전용 라이브 데모 위에서 촬영. 순서는 **머니샷 1 → 2 → 3 → 4**.
총 90초. 각 장면: 클릭 경로 · 자막 한 줄 · 소요 초.

## 사전 준비 (촬영 전)
- `DEMO_MODE=true` / `NEXT_PUBLIC_DEMO_MODE=true`로 배포 (→ `DEPLOY.md`)
- `python -m scripts.seed_demo` 실행 (멱등 — 항상 같은 화면)
- 브라우저 1280×720, 시크릿 창(로그인 없이 Admin 열리는 걸 보여주기 위함)
- 탭 미리 열어두기: `/admin` · `/admin/batches` · `/admin/translation` · `/admin/costs` · `/search`

---

## 장면 1 — 배치 파이프라인 (0:00–0:22, 22초)
- **클릭 경로**: `/admin` (대시보드) → 좌측 `🔄 배치 이력` (`/admin/batches`)
- **보여줄 것**: 최상단 배치 행 — `수집 A:609 · B1:8 → 디듀프 55건 → 발행 562`,
  상태 `완료`, 소요/토큰/비용 컬럼. 2단계 디듀프(URL SHA-256 + TF-IDF 0.85)를 자막으로.
- **자막**: “수집 → URL·TF-IDF 2단 디듀프 → 카드 발행. RSS 609 + GitHub 8 → 562건.”
- **팁**: 스크롤로 지난 배치 타임라인(일별)을 살짝 보여준다.

## 장면 2 — 번역 환각 게이트 (0:22–0:46, 24초)
- **클릭 경로**: 좌측 `🌐 번역 큐` (`/admin/translation`)
- **보여줄 것**: 검토 큐 카드 3건(유사도 **0.712 / 0.688 / 0.734** — 0.85 미달로 보류).
  카드 하나를 펼쳐 원문 · 한국어 번역 · **역번역** · 코사인 유사도 나란히.
  통과 카드는 0.887 / 0.912로 이미 공개됨을 대비로 언급.
- **자막**: “역번역 후 코사인 0.85 게이트. 0.88은 통과, 0.7대는 검토 큐로.”
- **팁**: 통과/미달 대비가 핵심. 미달 카드의 어색한 역번역(예: fish language)을 짧게 짚기.

## 장면 3 — 비용 통제 (0:46–1:08, 22초)
- **클릭 경로**: 좌측 `💰 비용 현황` (`/admin/costs`)
- **보여줄 것**: 일별 토큰·비용 막대 그래프, 이달 누적 **≈ $19.7 / 하드캡 $20**
  프로그레스 바(임박). 카드 1건 ≈ **$0.0037** 단가 언급.
- **자막**: “카드 1건 $0.0037. 월 $20 하드캡 도달 시 AI 루프 중단(무료 단계는 계속).”
- **팁**: 프로그레스 바가 거의 가득 찬 상태 → “한 배치만 더 돌면 캡”이라는 긴장감.

## 장면 4 — 한국어 검색 (1:08–1:30, 22초)
- **클릭 경로**: 공개 피드 `/` → 상단 검색 → `/search?q=검색`
  (또는 `한국어`, `프롬프트 캐싱`으로 재현율 시연)
- **보여줄 것**: nori 형태소 분석으로 ‘검색/검색어’가 같은 어간으로 묶여
  여러 카드가 재현되는 결과 목록.
- **자막**: “nori 형태소 색인으로 한국어 질의 재현율↑ — ‘검색’이 ‘검색어’까지 잡습니다.”
- **마무리**: 카드 상세로 들어가 원본 URL 노출(전문 미저장 원칙)을 1초 보여주며 종료.

## 자막 스트립 (붙여넣기용)
1. `수집 → URL·TF-IDF 2단 디듀프 → 발행. RSS 609 + GitHub 8 → 562건.`
2. `역번역 후 코사인 0.85 게이트. 0.88 통과 · 0.7대 검토 큐.`
3. `카드 1건 $0.0037. 월 $20 하드캡 도달 시 AI 루프 중단.`
4. `nori 형태소 색인으로 한국어 검색 재현율↑.`

## 주의
- 데모는 읽기전용 — 토글/승인/삭제 버튼은 비활성(클릭 시도 장면은 넣지 말 것).
- 모든 수치는 `seed_demo.py`가 고정 → 재촬영해도 동일.
- 실제 Claude 호출은 없음(재생 모드) → 촬영 중 비용 0.
